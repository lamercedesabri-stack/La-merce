Configuración — primera vez

Para sincronizar los datos entre la compu y el celular, seguí estos 3 pasos. Todo es 100% gratis para siempre con Google.

1

Abrir Google Apps Script

Ir a script.google.com → "Nuevo proyecto" → borrá lo que hay → pegá todo el contenido del archivo Code.gs → Guardar (Ctrl+S)

2

Publicar como Web App

"Implementar" → "Nueva implementación" → Tipo: Aplicación web → Ejecutar como: Yo → Quién puede acceder: Cualquier persona → "Implementar" → Autorizar → Copiar la URL que aparece

3

Pegar la URL acá abajo

La URL termina en /exec. La misma URL funciona desde el celular y la compu.
